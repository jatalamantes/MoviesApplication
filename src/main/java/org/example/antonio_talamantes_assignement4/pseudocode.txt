Class Validation
* Each validate method returns an empty string if the method parameter matches the criteria
  represented by a regular expression or returns a string containing the error message about the
  validation if the input is invalid.

static validateTitle(title)
    Title can’t be empty and can contain letters, numbers, hyphens, colons, periods, and
    spaces. The first letter must be capitalized.

static validateYear(year)
    Year can’t be empty and only contains digits.

static validateSales(sales)
    Sales can’t be empty and can only contain digits. The decimal point is optional. The
    number of decimal places can be 0 or more.
